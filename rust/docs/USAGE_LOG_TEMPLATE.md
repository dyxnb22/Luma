# Personal usage log (14-day experiment)

This is an optional local worksheet for deciding which aliases or advanced surfaces deserve to
remain prominent. Keep it local and record no query text, titles, paths, clipboard contents, SSH
hosts, proxy URLs, or account identifiers.

```text
Date | Goal | Surface (global/slash/Hub/CLI/menu) | Canonical module/action | Outcome (success/cancel/no-results/permission/not-configured) | Friction | Help needed?
```

Review after 14 days or at least 20 representative tasks. Hide an alias only when it has near-zero
successful use, no irreplaceable task depends on it, and its discoverability or accidental-use
cost is observable. Hiding is preferred to deleting code so the surface remains recoverable.
